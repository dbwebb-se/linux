Functions
==============================
Examples in Bash

```sh

#!/bin/bash

# Example of simple function

function speak() {
    echo "Hi!"
}

speak
```
This echoes "Hi!"  

```sh

#!/bin/bash

# Example of function calls

function speakAgain() {
    echo "How are you?"
}

speak
speakAgain
```
This will echoe:  
```sh

"Hi!"
"How are you?"
```
